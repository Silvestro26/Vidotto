# Published Articles

This folder contains PDF files of the main articles by Giulio Vidotto published in international peer-reviewed journals.

## Access Publications

For a complete and updated list of all publications, please consult:
- [Google Scholar Profile](https://scholar.google.com/citations?user=1P6L8qQAAAAJ&hl=it)
- [ORCID Profile](https://orcid.org/0000-0002-3854-5087)

## File Organization

- PDF files are organized by year of publication
- Each file is named with the year and a short title for easy retrieval
- Regular updates based on new publications

## How to Access Files

1. Navigate to the published-articles folder
2. Click on the PDF file you wish to view
3. Download it by clicking the download icon or right-clicking "Save link as"
