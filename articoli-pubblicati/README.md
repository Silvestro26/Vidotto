## Accessing PDF Files

To access the PDF files of the published articles, please follow these steps:
1. Navigate to the `articoli-pubblicati` directory.
2. Click on the PDF file you want to view.
3. Download it by clicking on the download icon or by right-clicking and selecting 'Save Link As...'.
